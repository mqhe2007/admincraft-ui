# Admincraft-ui

适用于 Admincraft 的 UI 组件库。

## 使用

```shell
yarn add admincraft-ui
```

```js
import Admincraft from 'admincraft'
import admincraftUI from 'admincraft-ui'
Admincraft.add(admincraftUI)
new Admincraft({
  ...
})
```
